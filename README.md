## final project

### ini adalah implementasi automation testing dengan spesifikasi berikut : 

1. menggunakan java 21
2. graddle
3. selenium
4. cucumber

### aplikasi ini melakukan pengujian terhadap situs  https://www.demoblaze.com/ dengan spesifikasi test case sebagai berikut :
1. positive,negative (terbatas hanya pada input saja)
2. test pada halaman login dan logout
3. test pada halaman homepage
4. test pada halaman checkout

### aplikasi ini melakukan pengujian api terhadap situs   https://www.demoblaze.com/ dengan spesifikasi test case sebagai berikut :
1. positive, negative case
2. create user
3. get user
4. update user

adapun masukan untuk aplikasi ini adalah :
1. pada menu contact user berhasil send message dengan tanpa mengisi apapun pada yang seharusnya ini tidak dibolehkan
2. pada menu checkout user bisa langsung melakukan checkout tanpa adanya proses login terlebih dahulu
